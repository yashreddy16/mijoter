# Mijoter

**Mijoter** AI-powered global recipe assistant designed for Indian kitchens and beginner cooks.

Mijoter helps users discover and cook international dishes using ingredients commonly available in India. From Korean ramen and Japanese snacks to Mexican rice bowls and café-style pasta, the app generates beginner-friendly recipes with simple instructions, substitutions, cooking tips, and personalized cooking modes.

Built using aistudio.google.com

---

## 🚀 Features
- 🌍 Generate recipes for global cuisines
- 🇮🇳 Adapted for Indian kitchens
- 🧑‍🍳 Beginner-friendly cooking instructions
- 💸 Budget and hostel-friendly modes
- ⚡ Quick meal recommendations
- 🌱 Vegetarian recipe support
- 🔥 Spice-level customization
- 🎨 Modern premium UI
- ☁️ Secure Gemini API integration using Vercel Serverless Functions
- 📱 Responsive design for desktop and mobile

## How it works
User enters a dish or craving
Frontend sends request to secure backend
Backend calls Gemini API
AI generates a beginner-friendly recipe
Recipe is displayed in a clean modern UI
